# FUTURE_DS_02

## Task 2 - Customer Retention & Churn Analysis

This project analyzes customer churn patterns using the Telco Customer Churn dataset.  
The goal is to understand why customers leave a service and identify key retention drivers.

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Dataset
Telco Customer Churn Dataset

---

## Analysis Performed
The following analyses were conducted:

- Customer Churn Distribution
- Churn by Contract Type
- Monthly Charges vs Churn
- Tenure vs Churn

---

## Key Insights

- Customers with **month-to-month contracts** have the highest churn rate.
- Customers with **higher monthly charges** are more likely to churn.
- Customers with **longer tenure** tend to stay with the company.
